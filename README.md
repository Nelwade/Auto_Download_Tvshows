# Auto_Download_Tvshows
Automation project: The program scrapes a website to track the release of the latest episodes of your favorite series and then automatically donwloads the episodes from pirate bay. 

Required packages:
  1. Selenium
  2. BeautifulSoup4
  3. pyautogui
  4. pydirectinput
  5. win10toast
  6. Pillow
  7. OpenCv
